# Ahmed Nizhan Haikal | Personal Portfolio

A minimalist, clean, and sleek personal portfolio built with **Next.js**, **TypeScript**, and **Vanilla CSS**. This project is optimized for performance, accessibility, and seamless deployment on **Vercel**.

![Portfolio Preview](public/screenshot.png) *(Note: Add a screenshot of your site here once deployed!)*

## ✨ Features

-   **Modern Minimalist Design:** A premium "Deep Obsidian" aesthetic with subtle indigo accents.
-   **Fully Responsive:** Optimized for all devices, from mobile to desktop.
-   **Glassmorphism:** Elegant blurred navigation and UI elements.
-   **Data-Driven:** All content (Experience, Skills, Education, Awards) is centralized in a single file (`src/lib/data.ts`) for easy updates.
-   **Icon-Rich:** Professional iconography powered by **Lucide React**.
-   **Performance Optimized:** Fast load times, zero Layout Shift (CLS), and smooth staggered animations.
-   **Accessibility First:** Visible focus states and semantic HTML, respecting user motion preferences.

## 🛠️ Tech Stack

-   **Framework:** [Next.js](https://nextjs.org/) (App Router)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **Styling:** Vanilla CSS (CSS Modules)
-   **Icons:** [Lucide React](https://lucide.dev/)
-   **Deployment:** [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites

-   Node.js 18+ 
-   npm / pnpm / yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/InMyDream21/portofolio.git
    cd portofolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📝 How to Update Your Content

You don't need to touch any components to update your portfolio! Simply modify the data in:

`src/lib/data.ts`

Change your name, summary, experiences, or skills there, and the site will update automatically.

## 🌍 Deployment

The easiest way to deploy is using the **Vercel Platform**:

1.  Push your code to GitHub.
2.  Import your repository into [Vercel](https://vercel.com/new).
3.  Deploy!
